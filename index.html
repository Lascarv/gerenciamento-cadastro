SystemCad

Bem-vindo ao sistema de gerenciamento de cadastro!

Escolha como começar

O systemCad foi desenvolvido para gerenciar de uma maneira simples e completa as informações dos usuários em um banco de dados. O front-end foi construído com HTML, CSS e JavaScript, enquanto o back-end foi implementado em Java com Spring Boot, integrando-se a um banco de dados MySQL. O controle de versão foi realizado com Git e GitHub, garantindo organização e colaboração no desenvolvimento. Além disso, a documentação da API foi feita com Swagger, e o design da interface foi planejado no Figma.
